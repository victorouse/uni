# Getting started
Clone the repo:

```
$ git clone https://victorouse@bitbucket.org/cavr/web-app.git cd && $_
```

Make sure you have Node (0.10.x) and NPM installed.. then:

```
$ npm install
```

Run:

```
$ node server/server.js
```
